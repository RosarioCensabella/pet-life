# Pet Life — Release Notes v1.0

## Version

1.0.0+1

## Highlights

Pet Life v1.0 helps pet owners organize the daily care, documents and due dates of their pets.

## Included features

- Italian and English localization
- Onboarding with medical-veterinary disclaimer
- Multi-pet home
- Pet profile creation
- Pet profile editing
- Pet archive flow
- Pet dashboard
- Reminder creation
- Reminder completion
- Reminder postpone and skip actions
- Local notifications for reminders
- Notification permission management from Settings
- Upcoming reminders on Home
- Global calendar
- Calendar filtering by pet
- Local document archive per pet
- Document file picker
- Document opening
- Document deletion
- Settings screen
- Privacy Policy in-app
- Terms of Service in-app
- Medical-veterinary disclaimer in-app
- Local JSON data export
- Local data deletion
- Free/Premium subscription model readiness
- Informational Premium paywall

## Not included in v1.0

- No medical diagnosis
- No triage
- No prescriptions
- No treatment recommendations
- No dosage calculations
- No cloud account
- No cloud sync
- No real store purchases yet
- No restore purchases yet
- No analytics
- No crash reporting

## Compliance note

Pet Life is an organizer, diary and document archive.  
It does not replace the veterinarian.
